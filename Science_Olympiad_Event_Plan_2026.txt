SECTION 5: SAFETY & EMERGENCY PROTOCOLS
================================================================================

MEDICAL EMERGENCY
------------------
1. First responder (nearest volunteer) assesses situation
2. Radio call to Command Center: "MEDICAL EMERGENCY [location]"
3. Medical Officer responds immediately
4. If serious: Call University Health Services / 112
5. Isolate area, clear crowd
6. Document incident
7. Notify parents (if student) via school liaison
8. Continue event if possible, or pause if necessary

FIRE EMERGENCY
---------------
1. Nearest person activates fire alarm
2. Evacuate building via nearest exit
3. Assembly at designated point
4. Roll call by volunteer team leaders
5. Fire service notification
6. Do NOT re-enter until cleared

SECURITY INCIDENT
------------------
1. Alert nearest security personnel
2. Radio call to Command Center
3. Security Lead assesses & responds
4. Isolate incident area
5. Document & report
6. Notify police if criminal activity

MISSING PERSON
---------------
1. Report to Command Center immediately
2. Last known location identified
3. Volunteer search team deployed
4. PA announcement (if appropriate)
5. Check restroom, lost & found, parking
6. Notify parents after 15 minutes
7. Security camera review if available

EQUIPMENT FAILURE
------------------
1. Report to Lab Supervisor
2. Attempt backup equipment
3. If unresolvable: Pause event, move to backup station
4. Document for post-event review
5. Adjust timing if needed
